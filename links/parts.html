Part Descriptions

The ESP32 is a low-cost, low-power, microcontroller. It has integrated Wi-Fi and Bluetooth. This microcontroller is programmed using the same language as Arduino C++. It is compatible with a plethora of other devices, such as: potentiometers, LCD Displays, Ultrasonic Sensors, and LEDs.



An ESP32 breakout board is a compact electronic module designed to provide easy access and interface to the ESP32 microcontroller.



The breadboard is a component that allows for other devices to be easily connected and rearranged without the use of soldering. Instead, it uses jumper cables.



Potentiometers are analog input devices that are often abbreviated as pots. They are used for controlling the electrical voltage coming through a circuit.



The SSD1306 OLED display is a monochrome display using IC technology. It is 128 x 64 pixels. This display also allows for individual configuration of each pixel.



Buttons are similar to on and off switches. They control whether or not power is being sent through them or if it is being stopped.



The male to male jumper wires alows us to connect the ESP32 to the breadboard and connect all our other componets the the ESP32 so they can comunicate with eachother.



The Solid Core Wires allow us to neatly connect our componets on the breadboard that dont need to be directly connected to the ESP32.



The Wire Stripper allows us to strip the sheilding off the Solid Core Wires so we can connect them to our breadboard and the components on them.



The USB A to Micro USB allows for the use of a USB cable to connect the ESP32 to the computer so we are able to load the code onto the ESP32.
